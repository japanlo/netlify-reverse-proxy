Netlify Reverse Proxy

A simple reverse proxy that can be used to access APIs and websites.

How to Use

The proxy works by adding /proxy/ or /httpproxy/ before the domain you want to access.

Examples:

For HTTPS:

/proxy/api.openai.com/v1/chat/completions

This will proxy to https://api.openai.com/v1/chat/completions

For HTTP:

/httpproxy/example.com/some/path

This will proxy to http://example.com/some/path

Special Routes:

/chat

This will proxy to https://chat.openai.com/backend-api/conversation

/openai

This will proxy to https://api.openai.com

Example Use Cases

  • Access OpenAI API without CORS issues
  • GitHub downloads acceleration
  • Access to restricted content
  • Content delivery optimization
